# FlashLearn – Android Flashcard & Trivia Learning App

FlashLearn is an Android study application that helps users learn through interactive flashcards and quiz-based trivia modes. The app emphasizes clean UI/UX, offline functionality, and structured application logic.

## Project Overview
FlashLearn allows users to study using two main modes:
- **Flashcard Mode** – swipe-based cards with tap-to-flip interactions
- **Trivia Mode** – multiple-choice quizzes with instant feedback and animations

The application runs fully offline and focuses on responsiveness, usability, and intuitive navigation.

## My Contributions
This project was completed as a **team-based academic assignment**.  
My contributions included:
- Android application development using **Java and Kotlin**

- Logic for quiz interactions, answer validation, and user feedback

## Key Features
- Swipe-based flashcards with progress tracking
- Multiple-choice trivia with animated feedback
- Clean, modern Android UI
- Fully offline functionality (no APIs or external services)

## Technologies Used
- **Languages:** Java, Kotlin
- **Tools:** Android Studio, Git, GitHub
- **UI:** ConstraintLayout, XML-based animations
- **Architecture:** Activity-based navigation, repository pattern

## Installation
1. Clone the repository  
2. Open the project in Android Studio  
3. Sync Gradle and run on an emulator or physical device  

> Note: This project does not require API keys or internet access.
